# ⭐ Starla's Diary

An AI-powered diary that analyzes your emotions, tracks your mood over time, and provides insights through Starla, an animated star companion.

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🤖 **AI Sentiment Analysis** | Detects emotions using Groq AI (Llama 3.3) |
| 📝 **Diary Entries** | Create, edit, and delete entries |
| 📊 **Emotional Insights** | 7-day rolling positivity score and mood trends |
| 📅 **Mood Calendar** | Color-coded timeline of your emotions |
| ⭐ **Starla Avatar** | Animated star companion that responds to your mood |
| 🔐 **Secure** | Google authentication with Row Level Security |

## 🛠️ Tech Stack

- **Frontend**: React 18, TypeScript, Tailwind CSS, Vite
- **Backend**: Supabase (PostgreSQL, Auth, Edge Functions)
- **AI**: Groq Cloud (Llama 3.3 70B)
- **Deployment**: Vercel

## 📦 Quick Start

### Prerequisites
- Node.js 18+
- Supabase account (free)
- Groq API key (free)
